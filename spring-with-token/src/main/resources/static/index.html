JWT Demo — Spring Boot

Status: nicht eingeloggt

1. Login — POST /auth/login

2. Geschützter Endpunkt — GET /api/hello

Erreichbar für alle eingeloggten Benutzer.

3. Admin-Endpunkt — GET /api/admin

Nur für Benutzer mit Rolle ROLE_ADMIN. Mit "user" kommt HTTP 403.

4. Frankfurt-Endpunkt — GET /api/frankfurt

Nur für Benutzer mit Wohnort Frankfurt (Custom JWT-Claim).
admin wohnt in Frankfurt ✓    user wohnt in Berlin ✗ (HTTP 403)
